
| 문제 유형                     | 최종 층 활성화 함수 | 손실 함수                     | 예시                                                       |
|-------------------------------|---------------------|------------------------------|-------------------------------------------------------------|
| 이진 분류                     | sigmoid             | binary_crossentropy          | 개 vs 고양이, 감정 분석(긍정/부정)                           |
| 다중 클래스, 단일 레이블 분류 | softmax             | categorical_crossentropy     | MNIST: 10개의 클래스, 단일 레이블 (한 번에 하나의 숫자 예측) |
| 다중 클래스, 다중 레이블 분류 | sigmoid             | binary_crossentropy          | 뉴스 태그 분류: 하나의 블로그에 여러 태그가 포함될 수 있음   |
| 임의의 값에 대한 회귀         | 없음                | mse                          | 집값 예측 (정수 또는 실수 예측)                              |
| 0과 1 사이 값에 대한 회귀     | sigmoid             | mse 또는 binary_crossentropy | 엔진 상태 평가: 0은 고장, 1은 새 것                          |



2. **최종 층 활성화 함수**: 각 문제 유형에 적합한 **최종 층의 활성화 함수**를 나타냄
   - **Sigmoid**: 이진 분류에서 사용되는 활성화 함수로, 출력이 0과 1 사이의 확률로 변환
   - **Softmax**: 다중 클래스 문제에서 사용되는 활성화 함수로, 클래스별 확률 분포를 반환
   - **None (없음)**: 회귀 문제에서는 활성화 함수가 필요하지 않음, 값 자체를 예측하기 때문!!!!!

3. **손실 함수**: 모델이 예측한 값과 실제 값 간의 오차를 측정하는 **손실 함수**를 나타냄
   - **binary_crossentropy**: 이진 분류와 다중 레이블 분류에서 사용되며, 이진 클래스에 대한 오차를 계산
   - **categorical_crossentropy**: 다중 클래스 분류에서 사용되며, 예측한 확률과 실제 레이블 간의 오차를 계산
   - **mse (Mean Squared Error, 평균 제곱 오차)**: 회귀 문제에서 주로 사용되며, 예측 값과 실제 값 사이의 차이를 제곱한 후 평균을 계산하는 방식
